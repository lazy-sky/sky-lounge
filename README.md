## 화면 예시(기능 설명)

### 기능 목록

- 게시글 열람
  - 최신순/좋아요순 정렬
  - 태그 필터링
- 게시글 작성(회원)
  - 글 작성
  - 사진 첨부
  - 태그 추가
- 게시글 좋아요(회원)

- 댓글 열람
- 댓글 작성(회원)

- 익명 채팅 참여
  - 게스트 계정 자동 생성
  - 사진 및 이모티콘 첨부
  - 메세지 리액션

- 회원가입 및 로그인
  - 로그인 시 '내가 쓴 글'과 '내가 쓴 댓글' 열람
  - 랜덤 닉네임 생성 가능
